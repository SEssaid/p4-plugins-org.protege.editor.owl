OWL Editor

This is one of the core Protégé plugins and is responsible for creating the core owl editing environment. This update should only be used by users running the 217 release of Protege that came out on the 22nd of February.

This is a critical update for users of Protégé 4.1 Build 217.

Version info

v4.1.0 Update Build 217 - 24 Feb, 2010

  • Fixed an issue with the way in which RDF files are saved. This is an important update as there is a small chance of data loss with the previous algorithm.
  • Added the ability to specify an IRI value for an annotation even if there is no associated entity.
